# Nostr Basic Event Creation

This Python script demonstrates the core Nostr event creation process. It shows how to generate a signed Nostr event (a text note) using a private key, which is fundamental for any Nostr client.

## Language

`python`

## How to Run

1. Install the `secp256k1` library: `pip install secp256k1`
2. Replace `YOUR_32_BYTE_HEX_PRIVATE_KEY` with your actual 32-byte hex private key.
3. Run the script: `python nostr_event_creator.py`
